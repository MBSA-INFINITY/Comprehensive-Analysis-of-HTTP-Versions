# Comprehensive Analysis of HTTP Versions

This project provides a comprehensive analysis of HTTP versions 1.1, 2, and 3. It uses Go to initialize HTTP/1.1, HTTP/2, and HTTP/3 servers on different ports. A Python-based client, utilizing the required libraries, is used to analyze and compare the following aspects:

- **HTTP/1.1:** Pipelining behavior and its limitations
- **HTTP/2:** Multiplexing capabilities and parallelization of requests
- **HTTP/3:** QUIC protocol features and request handling

The analysis focuses on the parallelization of requests, protocol-specific features, and other important parameters that distinguish each HTTP version. The project aims to provide insights into the performance and behavior of each protocol under various conditions.
